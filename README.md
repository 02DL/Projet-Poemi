# Getting started
## Prerequisites
Own a local server

## Installation
1. Clone the repo
2. Install Database 
` localhost/poemi/create_database.php `
3. Run the website
` localhost/poemi/index.php `

## Usage
The database of the repo is empty, you have to input your own data while using the website
